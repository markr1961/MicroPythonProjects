### RP2350 LED Matrix board
Code for the [WaveShare RP2350-matrix](https://www.waveshare.com/wiki/RP2350-Matrix) dev board

####Pins
- LED_DIN (GPIO25) Q1 (DMG1012T-7) drives WS2812B DIN to 5V. 
- RX/TX on header
- Boot/Key on IO0

#### [QMI8658](https://qstcorp.com/upload/pdf/202202/QMI8658C%20datasheet%20rev%200.9.pdf) QST 6-axis sensor (3-axis accel, 3-axis gyro)
Arduino and Circuit Python drivers available.  
Link to my page: https://jutland.atlassian.net/wiki/x/BwAVHQ

At least partially compatible with BMI160.
